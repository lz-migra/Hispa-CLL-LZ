Ayúdame a llenar unas plantillas con los datos de la Pagina Web Actual, puedes usar pistas la Web Actual.

Solo cuando te pida por escrito alguna de las siguientes plantillas llenaras la que corresponda.

- ID cliente: 
- Nro de orden: 
- Solicitud:

- Orden ID:

- Solicitud: 
Consideraciones para "Envios"
1. El Campo "- Canal: " Siempre es CC.

- ID del cliente:

- Provincia: 
- Número de reparto:

- ID o FOI: (Este lo encuentras al lado izquierdo de "tipo")

- Proveedor:

 1. El dato "ID Oferta" no es "- ID o FOI: "
 2. El dato "ID" Corresponde a "- ID o FOI: "

"Recarga" rellenas la siguiente plantilla

- Order code: 
- ID o FOI:  (Este lo encuentras al lado izquierdo de "tipo". En caso de haber varios solo el primero)
- Status: (En caso de haber varios solo el primero)
- Operador: 
- Solicitud: 

Consideraciones para todas las plantillas:

3. Solo podrás usar los datos de la Pagina Web Actual.
4. Si un dato no tiene valor o no se encuentra se deja "(No especifica)", a excepción del campo "- Solicitud: " , este lo dejaras sin texto.
5. Buscar el valor mas cercano al dato.
6. Extrae la mayor cantidad de datos posible, pero sin agregar datos no solicitados.
7. Solo con la primera orden presentada en la pagina.
8. Todo en formato de lista.
9. No puedes combinar las plantillas.
